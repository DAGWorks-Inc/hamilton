==================================
plugins.h_narhwals.NarwhalsAdapter
==================================

Provides a convenience wrapper for the `Narwhals` library; use the Narwhals decorator
underneath. Must have `Narwhals` installed to use it:

`pip install "sf-hamilton[narwhals]"`
